Positives
  - Language is written entirely in Java
  - Available at no cost for academic use
  - Uses Rete algorithm
    - A very efficient mechanism for solving the
      difficult "many-to-many" matching problem
  - Jess features include:
    - Backwards chaining
    - Working memory queries
    - Direct manipulation of java objects
  - Lots of online documentation
  - Eclipse-based rule development environment
